# 부동산 실거래가 동향 검색 서비스

## 📝 프로젝트 소개
지역별 월세, 전세, 매매 실거래가 정보를 제공하는 웹 서비스

## 🛠 기술 스택
### Backend
- Java 17
- Spring Boot 3.2.0
- Spring Data JPA
- H2 Database
- Spring Security

### Frontend
- React 18
- TypeScript
- Material-UI
- Axios

## 🚀 시작하기

### 필요 조건
- JDK 17+
- Node.js 18+

### 설치 및 실행

#### Backend
```bash
./gradlew build
./gradlew bootRun
```

#### Frontend
```bash
cd frontend
npm install
npm start
```

## 📁 프로젝트 구조
```
real-estate-tracker/

├── src/main/resources/     # 리소스 파일
```
